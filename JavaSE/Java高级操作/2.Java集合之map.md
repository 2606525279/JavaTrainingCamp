# [Map集合](https://docs.oracle.com/javase/8/docs/api/java/util/Map.html)

Map实现类用于保存具有映射关系的数据。Map保存的每项数据都是key-value对，也就是由key和value两个值组成。Map里的key是不可重复的，key用户标识集合里的每项数据。其中最主要的实现类是HashMap和TreeMap。
