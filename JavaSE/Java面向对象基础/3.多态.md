# 多态

## 学习目标
- 理解面向对象的多态性
- 掌握Java中用代码表达多态性
- 掌握对象的向上(向下)转型
- 掌握使用instanceof判断对象类型
- 掌握方法重写(Override),熟悉方法重载(Overload)

1. 深入理解重写与重载的概念，并写代码说明。
2. 写代码实现以下要求：
* 对于打印机来说，最基本的功能是打印。
* 对于彩色打印机来说，它可以打印彩色纸。
* 对于黑白打印机来说，就只能够打印黑白纸了。
